Customer Segmentation Using K-Means Clustering

This project uses the K-Means clustering algorithm in Python to segment customers based on their annual income and spending score. The analysis helps identify different customer groups such as high-value customers, average customers, and low-spending customers. The project includes data exploration, visualization, the Elbow Method for selecting the number of clusters, and a scatter plot showing the customer segments. Tools used include Pandas, Matplotlib, and Scikit-learn.
